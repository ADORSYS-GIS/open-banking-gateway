# open-banking-gateway
Provides tools, adapter and connector for transparent access to open banking apis. The initial effort focuses on the connectivity to banks that implement the European PSD2 directive either through one of the common market initiatives like :   * [The Berlin Group NextGenPSD2](https://www.berlin-group.org/psd2-access-to-bank-accounts)   * [The Open Banking UK] (https://www.openbanking.org.uk/)   * [The Polish PSD2 API] (https://polishapi.org/en/)  Or even proprietary bank api like [the INGâ€™s PSD2 API] (https://developer.ing.com/openbanking/)
